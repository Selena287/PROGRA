EXPLICACION DEL PROGRAMA:
Este programa tiene un menu interactivo que permite realizar las siguientes acciones con los datos que se ingresen.
Numeros impares: se guardan en una pila.
Numeros pares: se guardan en una cola.
Numeros negativos: se guardan en una lista enlazada.
Peek: permite ver el elemento superior de la pila.
Ordenar: ordena los numeros de mayor a menor en la lista.
Estadisticas: realiza un conteo de los numeros pares e inmpares que se han ingresado.
Gestion de Memoria: libera la memoria reservada para evitar leaks.
