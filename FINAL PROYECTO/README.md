Esto es un sistema que simula una red social academica la cual fue desarrollada en C++, se utilizo estructuras que fueron implementadas manualmente. Tambien se incluyo en Java un benchmark que se utilizo para comparar los rendimientos con las estructuras nativas.

En C++ la tabla Hash se utilizo para la busqueda de estudiantes por medio de ID, el arbol AVL se utilizo para el ranking academico por medio de skill_score, el grafo permitio las conexiones entre estudiantes por medio de BFS Y DFS, la lista enlazada permite colocar un proyecto nuevo a los estudiantes y en la cola se manejan las solicitudes de conexiones.

Las Funciones utilizadas fueron:

1. Cargar estudiantes desde CSV  
2. Registrar estudiante  
3. Buscar estudiante  
4. Eliminar estudiante  
5. Agregar proyecto  
6. Mostrar proyectos  
7. Conectar estudiantes  
8. Verificar conexión  
9. Mostrar conexiones  
10. BFS  
11. DFS  
12. Ranking  
13. Estadísticas  
14. Benchmark  
15. Salir
